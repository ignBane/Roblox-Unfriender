# Roblox-Unfriender
Unfriends everyone on your roblox friends list (Can ignore)

# Features
- Ignore List<br>
- Free

# Used-Modules
- [Colorama](https://pypi.org/project/colorama/) **pip install colorama**<br>
- [Requests](https://requests.readthedocs.io/en/master/) **pip install requests**<br>
- [JSON](https://docs.python.org/3/library/json.html) **pip install json**<br>
